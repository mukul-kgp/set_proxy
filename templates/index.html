Proxy Master

One Script to Set It All!

Download

set_proxy-master.sh

Instructions

  1. Open the downloads directory in a terminal window.
  2. Execute sudo bash set_proxy-master.sh and enter the password.
  3. Enter the proxy host and port on prompt. (172.16.2.30:8080 for KGP)

Going Home?

  • Execute sudo bash set_proxy-master.sh --unset and enter the password.

Still Need To Manually Set Proxy On:

  • Some Users might need to change Settings->Network->Proxy to Manual from Off.
